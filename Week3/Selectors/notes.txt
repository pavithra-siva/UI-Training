Combinators Selectors:(relationship between the elements)
    -Descendent Selectors - all element inside the first selectors(space)
    Eg:div p{ select all paragraph element inside the div}, ul.foods li{} <ul class="foods"><li></li></ul>
    -Child Selectors -Immediate child (>)
    -General siblings Selectors- selects all elements that are next siblings of a specified element (~)
    -Adjacent siblings Selectors- The + selector is used to select an element that is directly after another specific element.</p>
